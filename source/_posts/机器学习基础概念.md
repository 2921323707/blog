# 机器学习基础概念

机器学习是人工智能的核心分支，让计算机能够从数据中学习并做出预测

## 三大类型

### 1. 监督学习
使用标注数据训练模型，预测未知数据。

**示例：**
- 分类：识别邮件是否为垃圾邮件
- 回归：预测房价

### 2. 无监督学习
从未标注数据中发现模式。

**示例：**
- 聚类：客户分群
- 降维：数据可视化

### 3. 强化学习
通过与环境交互，学习最优策略。

**示例：**
- 游戏AI
- 自动驾驶

## 基本流程

```python
# 1. 数据准备
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2
)

# 2. 选择模型
from sklearn.ensemble import RandomForestClassifier
model = RandomForestClassifier()

# 3. 训练
model.fit(X_train, y_train)

# 4. 预测
predictions = model.predict(X_test)

# 5. 评估
from sklearn.metrics import accuracy_score
accuracy = accuracy_score(y_test, predictions)
```

## 常用算法

- **线性回归**：预测连续值
- **决策树**：易于解释的分类
- **随机森林**：集成学习，提高准确率
- **K-means**：无监督聚类
- **神经网络**：深度学习基础

## 关键概念

- **特征工程**：选择和转换输入特征
- **过拟合**：模型过度记忆训练数据
- **交叉验证**：评估模型泛化能力
- **超参数调优**：优化模型性能

## 总结

机器学习通过算法从数据中学习规律，广泛应用于推荐系统、图像识别、自然语言处理等领域。掌握基础概念是深入学习的第一步！
